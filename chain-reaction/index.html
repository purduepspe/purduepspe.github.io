What is Rube Goldberg?

Rube Goldberg Machine contests challenge participants to accomplish a simple task in the most complex, convoluted way possible using many simple machines - useless on their own - to construct a bigger machine capable of completing the task at hand. The task changes every year, but it is usually something easy for humans to do but possibly quite difficult for a simple machine. To read more, see this link.
